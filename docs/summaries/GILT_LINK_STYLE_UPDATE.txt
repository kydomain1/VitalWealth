========================================
📝 CSS代码详情
========================================

```css
.article-content a {
    color: #0066cc;              /* 蓝色 */
    text-decoration: none;       /* 移除默认下划线 */
    font-weight: 600;            /* 粗体 */
    border-bottom: 2px solid #0066cc;  /* 自定义下划线 */
    padding-bottom: 2px;
    transition: all 0.3s ease;   /* 平滑过渡 */
}

.article-content a:hover {
    color: #004499;              /* 深蓝色 */
    border-bottom-color: #004499;
    background-color: rgba(0, 102, 204, 0.1);  /* 浅蓝背景 */
    padding: 2px 4px;
    border-radius: 2px;          /* 圆角 */
}

.article-content a:active {
    transform: translateY(1px);  /* 点击下移 */
}
```
