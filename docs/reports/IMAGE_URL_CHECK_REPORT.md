## URL格式分析

### 标准格式
```
https://images.unsplash.com/photo-{ID}?w={width}&h={height}&fit=crop
```

### 参数说明
- **w**: 宽度（width）
- **h**: 高度（height）
- **fit**: 裁剪方式（crop）

### 使用的尺寸

| 用途 | 尺寸 | 数量 |
|------|------|------|
| 文章主图 | 800×600 | 5 |
| 内容图片 | 800×500 | 15 |
| 产品图片 | 400×400 | 15 |
