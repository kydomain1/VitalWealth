📝 常见问题
========================================

Q1: 推送时要求输入用户名密码？
A: 使用Personal Access Token作为密码

Q2: 推送被拒绝（rejected）？
A: 先执行 git pull origin main，再推送

Q3: 文件太大上传失败？
A: GitHub单文件限制100MB

Q4: 想修改提交信息？
A: 执行 git commit --amend -m "新的提交信息"

Q5: 推送后发现有错误？
A: 修改后执行：
   git add .
   git commit -m "Fix: 描述修复内容"
   git push
